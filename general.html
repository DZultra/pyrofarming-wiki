General Information

What are the contents of the Plugin?

PyroFarming is an exciting plugin that introduces new plants, bushes and ways for you to make money. It is currently based on two main topics: Plants and Bushes. The purpose of this plugin is to add a new way of making money and does not add any items which have an effect to vanilla itmes. All items which you craft or buy are only useful within the plugin's contents.

Furthermore, the plugin also adds new commands which are listed with their description here:

Player Commands

Command Description
/farm View a list of the PyroFarming commands
/farm menu Opens the main farming menu
/farm help <args> Opens a book that provides some basic information about the Plugin
/farm codex <plants/bushes> Opens the PyroFarming Codex. There you can see all of the Plants and Bushes the Plugin has to offer
/farm elysium Shows your total Elysium, the custom currency for the Plugin's Skill Archive
/farm market <permanent/rotating> Opens the Market, where you can buy seeds from the permanent Shop or the rotating Shop, which offers seeds at a discount
/farm shop Opens the PyroFarming Shop, where you can sell your harvested plants
/farm placeditems Provides a list of all Growstations and Bushes the player, running the command, has placed with there coordinates
/farm friends <args> Add and remove Friends which can access your Growstations and harvest your Bushes.

Admin Commands

Command Description
/farm givegrowstation <player> Give a Growstation to the specified player
/farm giveplantharvestable <plant> <harvestId> Give the specified player the specified plant harvestate item
/farm givebushharvestable <bush> <harvestId> Give the specified player the specified bush harvestate item
/farm giveseed <seed> <player> Give the specified player the specified seed
/farm givebush <bush> <player> Give the specified player the specified bush
/farm reload Reloads the entire Plugin
